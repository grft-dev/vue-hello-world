# Vue + Vite

This template provides a minimal setup to get Vue 3 working in Vite with HMR.

It is the starting point for the Graftcode quick-start guide:
[Connect a Vue frontend to a backend](https://github.com/grft-dev/graftcode-quick-start-guide/blob/main/1-connect-frontend-to-backend/vue.md).

## Getting started

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (typically [http://localhost:5173](http://localhost:5173)).
